Citações

Citações completas

Citações completas são distribuidas em o texto que foi digo ou escrito por alguém e o link do livro ou pessoa que o fez. O link é adicionado com a tag BLOCKQUOTE e o cite dentro do blockquote

Citações simples

são determinadas pela tag q exemplo: fulano disse que o neymar joga muito

Código fonte/pré-formatação

O código fonte CODE é usado para igalar as letras do texto e manpular o texto

O PRE é usado para pre formatar o texto desejado e configurar o texto.
