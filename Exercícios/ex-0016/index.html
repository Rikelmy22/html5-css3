Tipos de formatações em html

Texto marcado

podemos criar um texto marcado usando a tag MARK

E no outro parágrafo temos outro texto marcado no final coloquei cor aqui usando a tag style no cabeçalho

Texto grande e pequeno

Estamos criando texto grande e pequeno nesse parágrafo o pequeno é a tag SMALL

Texto deletado

Podemos colocar um texto que foi inserido mas não deve ser considerado(Usando a tag del)

Texto inserido

Podemos usar a tag INS para dizer que o texto só foi inserido depois

Existe também o texto sublinhado com a tag U (não é semântica).

Texto sobrescrito

para adicionar as coisas relacionadas a números elevados e contas usamos a tag sup, por exemplo:X 20+3

exemplo da molécula de água H2O exemplo: H 2O USANDO A TAG SUB
